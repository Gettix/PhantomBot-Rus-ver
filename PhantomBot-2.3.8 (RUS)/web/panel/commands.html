Панель Команд

(sender) Пользователь, который вызвал команду.
(@sender) Пользователь, который вызвал команду (Пользователь будет отмечен через @).
(baresender) Пользователь, который вызвал команду в нижнем регистре.
(touser) Заменяется аргументом, который вы напишете после команды. Если пользователь, вызвавшего команду, ничего не напишет, то сработает sender.
(#) Случайное число
(1), (2), (3) и т.д Аргументы для команд(ы).
(age) Говорит как долго Пользователь зарегистрирован на Twitch.
(random) Случайный Пользователь в чате.
(pointname) Текущее количество Очков.
(uptime) Сколько стрим идет (uptime).
(game) Текущая Игра на стриме.
(status) Текущее Название стрима.
(price) Текущая цена за команду.
(pay) Плата за команду.
(onlineonly) Команда будет работать Только когда стрим Он-Лайн.
(offlineonly) Команда будет работать Только когда стрим Офф-Лайн.
(viewers) Текущее количество зрителей.
(count) Количество времени когда команда была использована.
(echo) Скажет любое сообщение как бот.
(1=) Аргумент 1. Добавить имя тега на Ваш выбор после = если аргумент пуст.
(countdown=) Добавьте дату от которой делать отсчет. Пример: (countdown=Июль 2 2016 7:00).
(downtime) Время с последнего стрима.
(readfile filename.txt) Читает содержимое txt файла. Файл должен находиться в папке аддонов бота.
(customapijson URL key | {literal string} ...) Удаленно вызвать URL который вернет JSON. key или object.key или object.object.key для доступа к JSON файлу.
(command [command_name]) Вызвать команду. Должен быть первым элементом в команде. Не добавляйте "!" перед command_name
(customapi [requesturl]) Удаленно вызвать URL
